\# Speech to Text Project (Python)



\## Overview

This is a simple Python project that converts live human speech into text.

The program captures audio from the microphone and converts it into text using

Google Speech Recognition.



\## Features

\- Takes real-time voice input

\- Converts speech to text

\- Displays recognized text in the terminal

\- Handles unclear speech and API errors



\## Technologies Used

\- Python

\- SpeechRecognition

\- PyAudio



\## How to Run the Project



1\. Open terminal and go to the project folder

2\. Activate virtual environment

3\. Install required libraries:

&nbsp;  pip install -r requirements.txt

4\. Run the program:

&nbsp;  python speech\_to\_text\_demo.py

5\. Speak into the microphone when prompted



\## Output

The spoken words are displayed as text on the screen.



\## Use Case

This project can be extended for:

\- Voice assistants

\- Speech-based applications

\- Accessibility tools

\- AI and NLP projects
